# Авто-постеры

Скрипт для автоматической генерации постеров автомобилей по данным с [automobile-catalog.com](https://www.automobile-catalog.com/): выбор марки и модели, парсинг характеристик, генерация постера в стиле референса (AUDI TT RS) и сохранение в PNG/JPG.

## Возможности

- **Парсинг каталога** — загрузка списка моделей по марке и страниц с характеристиками
- **Устойчивость** — таймауты, повторы запросов, задержки; при недоступности сайта используется демо-набор данных
- **Постер в стиле референса** — тёмный фон, крупная типографика, блок ключевых характеристик (двигатель, мощность, разгон, привод и т.д.)
- **CLI** — удобный запуск с указанием марки, опционально модели и пути сохранения

## Требования

- Python 3.10+
- Доступ в интернет (для парсинга каталога)

## Установка

```bash
# Клонирование репозитория
git clone https://github.com/Artem9908/<ИМЯ_РЕПОЗИТОРИЯ>.git
cd <ИМЯ_РЕПОЗИТОРИЯ>

# Виртуальное окружение (рекомендуется)
python -m venv .venv
.venv\Scripts\activate   # Windows
# source .venv/bin/activate  # Linux/macOS

# Зависимости
pip install -r requirements.txt
```

Опционально скопируйте `.env.example` в `.env` и задайте переменные (таймауты, папка вывода, API-ключи при использовании нейросети).

## Использование

**Минимальный запуск (марка; модель подставится автоматически или демо):**
```bash
python main.py --brand Audi
```

**С указанием модели:**
```bash
python main.py --brand Audi --model "TT RS"
```

**С указанием выходного файла:**
```bash
python main.py -b BMW -m "M3" -o output/bmw_m3.png
```

**Подробный вывод (DEBUG):**
```bash
python main.py --brand Audi --model "TT RS" --verbose
```

Постер по умолчанию сохраняется в папку `output/` в формате PNG.

## Конфигурация (переменные окружения)

| Переменная | Описание | По умолчанию |
|------------|----------|--------------|
| `AUTO_CATALOG_BASE` | Базовый URL каталога | `https://www.automobile-catalog.com` |
| `AUTO_REQUEST_TIMEOUT` | Таймаут запроса (сек) | `15` |
| `AUTO_REQUEST_RETRIES` | Число повторов запроса | `3` |
| `AUTO_REQUEST_DELAY` | Задержка между запросами (сек) | `1.0` |
| `AUTO_OUTPUT_DIR` | Папка для постеров | `output` |
