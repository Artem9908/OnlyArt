# Авто-постеры

Скрипт для автоматической генерации постеров автомобилей по данным с [automobile-catalog.com](https://www.automobile-catalog.com/).

- Выбор марки (например, Audi)
- Поиск моделей и модификаций на сайте
- Парсинг характеристик
- Генерация постера в стиле референса (API нейросети или локально)
- Сохранение результата в PNG/JPG
